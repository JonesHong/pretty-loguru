## üöÄ Quick Start

1. **Clone examples**:
   ```bash
   git clone https://github.com/JonesHong/pretty-loguru.git
   cd pretty-loguru/examples
   ```

2. **Run first example**:
   ```bash
   cd 01_basics
   python simple_usage.py
   ```

3. **Check generated logs**:
   ```bash
   ls ./logs/
   cat ./logs/*.log
   ```
