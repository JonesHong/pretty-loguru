# pretty-loguru Documentation

This is the official documentation for pretty-loguru, built with VitePress.

## 🚀 Development

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
