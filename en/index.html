Pretty Loguru Documentation

繁體中文 | English

Pretty Loguru is an enhanced Loguru logging system that provides block-style logs, ASCII art titles, and integration with various frameworks, making logging more intuitive and visually appealing.

Features

  • Block-style log format for improved readability

  • ASCII art titles for enhanced visual effects

  • Goal-oriented logging approach with clear output targets (console, file)

  • Seamless integration with FastAPI and Uvicorn

  • Elegant log rotation and cleanup mechanism

  • Highly customizable formats and configurations

Installation

pip install pretty-loguru

Quick Usage Example

from pretty_loguru import create_logger

# 創建日誌實例
logger = create_logger("my_app")

# 輸出不同級別的日誌
logger.info("這是一條資訊日誌")
logger.warning("這是一條警告日誌")

# 使用區塊功能
logger.block(
    "系統狀態",
    [
        "數據庫連接: 正常",
        "緩存服務: 正常",
        "存儲空間: 76% 可用"
    ]
)
