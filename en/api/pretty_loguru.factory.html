pretty_loguru.factory packageÔÉÅ

Factory Module Entry Point

This module provides functionality for creating and managing Logger instances, including adding various custom methods. The factory pattern makes creating and configuring Logger instances simple and consistent.

pretty_loguru.factory.create_logger(name=None, service_tag=None, subdirectory=None, log_name_format=None, log_name_preset=None, timestamp_format=None, log_path=None, log_file_settings=None, custom_config=None, reuse_existing=False, start_cleaner=False, force_new_instance=True, console=None, level='INFO', rotation='20 MB')[source]ÔÉÅ

Create an effectively isolated logger instance

Parameters:

  • name (Optional[str]) ‚Äì The name of the logger instance, automatically generated if not provided

  • service_tag (Optional[str]) ‚Äì Service or module name, used to identify the log source

  • subdirectory (Optional[str]) ‚Äì Log subdirectory, used to categorize logs for different modules or functions

  • log_name_format (Optional[str]) ‚Äì Log file name format, can include variables such as {component_name}, {timestamp}, {date}, {time}, etc.

  • log_name_preset (Optional[Literal['default', 'daily', 'hourly', 'minute', 'simple', 'detailed']]) ‚Äì Default log file name format, optional values are ‚Äúdefault‚Äù, ‚Äúdaily‚Äù, ‚Äúhourly‚Äù, etc.

  • timestamp_format (Optional[str]) ‚Äì Timestamp format, used to customize the time display format

  • log_path (Union[str, Path, None]) ‚Äì Base path for logs, overrides the default log_path

  • log_file_settings (Optional[Dict[str, Any]]) ‚Äì Other settings for log files, such as compression, retention time, etc.

  • custom_config (Optional[Dict[str, Any]]) ‚Äì Custom log configuration, can include any parameters supported by configure_logger

  • reuse_existing (bool) ‚Äì Whether to reuse an existing instance with the same name, default is False

  • start_cleaner (bool) ‚Äì Whether to enable the log cleaner, default is False

  • force_new_instance (bool) ‚Äì Whether to force the creation of a new instance, default is True

  • console (Optional[Console]) ‚Äì The Rich Console instance to use, default is the globally shared instance

  • level (Literal['TRACE', 'DEBUG', 'INFO', 'SUCCESS', 'WARNING', 'ERROR', 'CRITICAL']) ‚Äì Log level, default is INFO

  • rotation (Union[str, int]) ‚Äì Log rotation settings, default is 20 MB
