pretty_loguru.formats package

Formatting module entry

This module provides all formatting features of Pretty Loguru, including block formatting, ASCII art, and FIGlet text. These features enhance the visual presentation of logs.

pretty_loguru.formats.print_block(*args, **kwargs)

Print block-style logs and write them to the log file

Parameters:

  • title – Block title

  • message_list – List of log contents

  • border_style – Block border color, default is “cyan”

  • log_level – Log level, default is “INFO”

  • logger_instance – Logger instance to use, if None, logs are not recorded

  • console – Rich console instance to use, if None, a new one is created

  • to_console_only – Whether to output only to the console, default is False

  • to_log_file_only – Whether to output only to the log file, default is False

  • _target_depth – Log stack depth, used to capture the correct call location

Return type:

Any
