pretty_loguru.integrations package

Integration module entry point

This module provides integration with third-party libraries and frameworks, enabling Pretty Loguru to seamlessly work with various popular Python tools and frameworks.

pretty_loguru.integrations.configure_uvicorn(logger_instance=None, level='INFO', logger_names=None)[source]

Configure Uvicorn logging to use Loguru formatting

This function modifies Uvicorn’s log output format to Loguru’s format, suitable for applications requiring unified log formatting.

Parameters:

  • logger_instance (Optional[Any]) – The logger instance to use, defaults to the default logger if None

  • level (Literal['TRACE', 'DEBUG', 'INFO', 'SUCCESS', 'WARNING', 'ERROR', 'CRITICAL']) – Log level, defaults to “INFO”

  • logger_names (Optional[List[str]]) – List of logger names to configure, defaults to Uvicorn-related loggers

Raises:

ImportError – If uvicorn is not installed

Return type:

None
