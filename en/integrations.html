Framework Integration

Pretty Loguru is designed to seamlessly integrate with popular Python frameworks. This section introduces how to integrate Pretty Loguru into different frameworks.

FastAPI Integration

Pretty Loguru provides full integration with FastAPI, including request logging, dependency injection, and error handling.

Basic Setup

from fastapi import FastAPI, Depends
from pretty_loguru import create_logger
from pretty_loguru.integrations.fastapi import setup_fastapi_logging, get_logger_dependency

# 創建 FastAPI 應用
app = FastAPI()

# 創建日誌實例
logger = create_logger("fastapi_app")

# 設置 FastAPI 日誌
setup_fastapi_logging(
    app,
    logger_instance=logger,
    middleware=True,
    custom_routes=True,
    log_request_body=True,
    log_response_body=False
)

# 基本路由
@app.get("/")
async def root():
    logger.info("處理首頁請求")
    return {"message": "Hello World"}

Dependency Injection

Use the get_logger_dependency function to provide a dedicated logger instance for each route:

# 創建路由特定的日誌依賴
users_logger = get_logger_dependency(name="users_api")

@app.get("/users/")
async def get_users(logger=Depends(users_logger)):
    logger.info("獲取用戶列表")
    return {"users": ["user1", "user2"]}

# 另一個範例
items_logger = get_logger_dependency(name="items_api", service_tag="inventory")

@app.get("/items/")
async def get_items(logger=Depends(items_logger)):
    logger.info("獲取商品列表")
    return {"items": ["item1", "item2"]}

Logging Middleware

When using setup_fastapi_logging to set up middleware, it will automatically log detailed information about all requests and responses:

# 自定義中間件設置
setup_fastapi_logging(
    app,
    logger_instance=logger,
    exclude_paths=["/health", "/metrics"],  # 排除某些路徑不記錄日誌
    exclude_methods=["OPTIONS"],            # 排除某些 HTTP 方法
    log_request_body=True,                  # 記錄請求體
    log_response_body=True,                 # 記錄響應體
)

Output Example:

2023-05-01 12:34:56 | INFO | Request [1682938496.123]: GET /users/ from 127.0.0.1:56789
2023-05-01 12:34:56 | INFO | Response [1682938496.123]: 200 in 0.015s

Custom Routes

Use the custom_routes=True parameter to enable more detailed route-level logging:

# 啟用自定義路由
setup_fastapi_logging(
    app,
    logger_instance=logger,
    custom_routes=True,
    log_request_body=True,
)

Uvicorn Integration

Pretty Loguru can take over Uvicorn’s logging system, ensuring all logs use the same format and handling.

Basic Setup

import uvicorn
from pretty_loguru import create_logger
from pretty_loguru.integrations.uvicorn import configure_uvicorn

# 創建日誌實例
logger = create_logger("uvicorn_app")

# 配置 Uvicorn 使用 Pretty Loguru
configure_uvicorn(logger_instance=logger, level="INFO")

# 啟動 Uvicorn
if __name__ == "__main__":
    uvicorn.run("app:app", host="0.0.0.0", port=8000)

Intercept Handler

The InterceptHandler class is used to intercept standard library logs and forward them to Loguru:

import logging
from pretty_loguru import create_logger
from pretty_loguru.integrations.uvicorn import InterceptHandler

# 創建日誌實例
logger = create_logger("my_app")

# 設置攔截處理器
logging.basicConfig(handlers=[InterceptHandler(logger_instance=logger)], level=0)

# 從此以後，所有透過標準庫的日誌都會被轉發給 Loguru
logging.info("這條日誌會被 Loguru 處理")

Integration with Other ASGI Servers

The integration method with other ASGI servers is similar:

from pretty_loguru import create_logger
from pretty_loguru.integrations.uvicorn import configure_uvicorn

# 創建日誌實例
logger = create_logger("asgi_app")

# 配置日誌
configure_uvicorn(
    logger_instance=logger,
    level="INFO",
    logger_names=["uvicorn.asgi", "uvicorn.access", "hypercorn", "daphne"]
)

Integration with Other Libraries

Pretty Loguru can integrate with any tool that uses the standard logging library:

SQLAlchemy Example

import logging
from pretty_loguru import create_logger
from pretty_loguru.integrations.uvicorn import InterceptHandler

# 創建日誌實例
logger = create_logger("sqlalchemy_app")

# 設置 SQLAlchemy 日誌
sql_logger = logging.getLogger("sqlalchemy.engine")
sql_logger.setLevel(logging.INFO)
sql_logger.addHandler(InterceptHandler(logger_instance=logger))

# 現在 SQLAlchemy 的日誌將使用 Pretty Loguru 格式

Django Example

# settings.py
LOGGING = {
    'version': 1,
    'disable_existing_loggers': False,
    'handlers': {
        'loguru': {
            'class': 'pretty_loguru.integrations.uvicorn.InterceptHandler',
        },
    },
    'root': {
        'handlers': ['loguru'],
        'level': 'INFO',
    },
}

Flask Example

from flask import Flask
import logging
from pretty_loguru import create_logger
from pretty_loguru.integrations.uvicorn import InterceptHandler

app = Flask(__name__)

# 創建日誌實例
logger = create_logger("flask_app")

# 配置 Flask 日誌
app.logger.handlers = [InterceptHandler(logger_instance=logger)]
app.logger.setLevel(logging.INFO)

# 路由範例
@app.route('/')
def hello():
    app.logger.info("處理首頁請求")
    return "Hello World!"
